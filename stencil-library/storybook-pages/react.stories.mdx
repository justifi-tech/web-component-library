# Usage

To use the web-components in a React app you need to add the CDN script and the link to the default styles to your `index.html` file:

<Source
  language="bash"
  dark
  format={false}
  code={dedent`
    <head>
      <link
        rel="stylesheet"
        href="https://cdn.jsdelivr.net/npm/@justifi/webcomponents@4.6.0/dist/webcomponents/webcomponents.css"
      />
      <script
        src="https://cdn.justifi.co/justifi-components/1.0.0/justifi-components.js"
      />
    </head>
  `}
/>

Then you can use the components as any other HTML element:

<Source
  language="jsx"
  dark
  format={false}
  code={dedent`
    <justifi-card-form
      single-line="true"
      validation-mode="onSubmit"
    />
`}
/>

# Usage details

## Tyepscript integration

To integrate with TypeScript, declare the component in the JSX.IntrinsicElements interface. This helps TypeScript understand the custom element, its attributes, and events.

<Source
  language="jsx"
  dark
  format={false}
  code={dedent`
    import React, { useState, DOMAttributes } from 'react';

    type CustomElement<T> = Partial<T & DOMAttributes<T> & { children?: React.ReactNode }>;

    declare global {
      namespace JSX {
        interface IntrinsicElements {
          'justifi-payment-form': CustomElement<any>; // Specify more specific types as needed
        }
      }
    }

`}
/>

## Listening to events

React allows attaching event listeners directly to JSX components. For custom events emitted by justifi-payment-form, use the addEventListener method within a useEffect hook or a ref callback.

Example of handling a custom event:

<Source
  language="jsx"
  dark
  format={false}
  code={dedent`
    import React, { useEffect, useRef } from 'react';

    export default function PaymentFormComponent() {
      const formRef = useRef(null);

      useEffect(() => {
        const form = formRef.current;
        if (form) {
          const handleSubmit = (event) => {
            // Handle the submit event
          };
          form.addEventListener('submitted', handleSubmit);

          // Cleanup
          return () => form.removeEventListener('submitted', handleSubmit);
        }
      }, []);

      return (
        <justifi-payment-form
          ref={formRef}
          client-id="your-client-id"
          email="customer@example.com"
          // ... other props
        />
      );
    }

`}
/>

## Calling Methods

In some scenarios, you might need to programmatically call methods on the justifi-payment-form web component from your React components. This can be done using React refs. Here’s a step-by-step guide:

### Step 1 Creating a Ref to the Component

First, create a ref in your React component that will be attached to the `justifi-payment-form` element. This ref will be used to access the web component and its methods.

<Source
  language="jsx"
  dark
  format={false}
  code={dedent`
    import React, { useRef } from 'react';

    export default function PaymentFormComponent() {
      const paymentFormRef = useRef(null);

      // Rest of your component code
    }

`}
/>

### Step 2 Attaching the Ref to the Web Component

Attach the created ref to the justifi-payment-form element in your JSX

<Source
  language="jsx"
  dark
  format={false}
  code={dedent`
  <justifi-payment-form
    ref={paymentFormRef}
    client-id="your-client-id"
    // ... other props
  />
`}
/>

### Step 3 Calling Methods on the component

You can now call methods on the justifi-payment-form component using the ref. It’s important to ensure that the component is mounted and the ref is set before attempting to call any methods. This is typically done inside event handlers or lifecycle methods like useEffect.

Example of calling a method:

<Source
  language="jsx"
  dark
  format={false}
  code={dedent`
    import React, { useRef, useEffect } from 'react';

    export default function PaymentFormComponent() {
      const paymentFormRef = useRef(null);

      useEffect(() => {
        if (paymentFormRef.current) {
          // Call a method on the web component
          paymentFormRef.current.someMethod();
        }
      }, []);

      return (
        <justifi-payment-form
          ref={paymentFormRef}
          client-id="your-client-id"
          // ... other props
        />
      );
    }

`}
/>

# Styling

Styling works similarly to the vanilla web components.

## Components styled with variables

The following components can only be styled through CSS variables:

- [BankAccountForm](?path=/docs/components-bankaccountform--docs)
- [CardForm](?path=/docs/components-cardform--docs)
- [PaymentForm](?path=/docs/components-paymentform--docs)

You can easily override the variables in a `<style>` tag in any place of your App, both in the `<head>` of your root `index.html` or at any component level.

<Source
  language='jsx'
  dark
  format={false}
  code={dedent`
    <YourComponent>
      <style>
        {\`:root {
          --jfi-variable: 'override value';
        }\`}
      </style>

      <JustifiPaymentForm />
    </YourComponent>
  `}
/>
