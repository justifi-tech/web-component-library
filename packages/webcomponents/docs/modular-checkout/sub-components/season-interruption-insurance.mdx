## Overview

Component to render a formated list of season interruption insurance for the requested account.

## Usage

<UsageSnippet
  title="Attach insurance upsell"
  code={`<justifi-modular-checkout
  account-id="acc_123"
  auth-token="wct_checkout"
  checkout-id="chk_456"
>
  <justifi-season-interruption-insurance
    slot="addons"
    primary-identity-first-name="Jamie"
    primary-identity-last-name="Rivera"
    primary-identity-email-address="jamie@example.com"
    primary-identity-state="MN"
    primary-identity-postal-code="55401"
    primary-identity-country="US"
    policy-attributes-insurable-amount="250000"
    policy-attributes-start-date="2024-12-01"
    policy-attributes-end-date="2025-03-31"
  />
  <justifi-modular-checkout-card-form slot="payment-methods" />
  <justifi-modular-checkout-summary slot="summary" />
</justifi-modular-checkout>`}
  language="html"
/>

### Events

- `insurance-updated`: Fires after the component successfully toggles coverage via the insurance API.
- `error-event`: Emits a `ComponentErrorEvent` if required props are missing or an API call fails.

- Both radio buttons render with native `<label>` wrappers for generous click targets and screen-reader friendly names.
