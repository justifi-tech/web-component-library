Welcome to the portable documentation bundle that ships with `@justifi/webcomponents`. Import these MDX files into any Docusaurus site to expose the same guidance we publish internally alongside Storybook.

## Usage

### Load the library with a script tag

```html
<head>
  <script
    async
    type="module"
    src="https://cdn.jsdelivr.net/npm/@justifi/webcomponents@latest/dist/webcomponents/webcomponents.esm.js"
  ></script>
</head>
```

### Install via npm or pnpm

```bash
npm install --save @justifi/webcomponents
# or
pnpm add @justifi/webcomponents
```

Then import the module for the component you need:

```javascript
import '@justifi/webcomponents/dist/module/justifi-checkout.js';
```

Once registered, render the custom element like any other HTML tag:

```html
<justifi-checkout auth-token="token" checkout-id="chk_123"></justifi-checkout>
```

## Styling primer

These components expose a layered set of CSS `::part` selectors so you can theme them globally or per component.

### How parts stack

1. **Base parts** such as `color`, `font-family`, and `background-color` define global primitives.
2. **Higher-level parts** like `text`, `input`, `button`, and `label` inherit from the base layer so typography and color cascade everywhere.
3. **Component-specific parts** (for example `button-primary` or `summary-card`) sit on top for precise overrides.

### Global styling examples

Apply a universal font:

```css
::part(font-family) {
  font-family: 'Inter', system-ui, sans-serif;
}
```

Centralize text color:

```css
::part(color) {
  color: #1d1b2f;
}
```

### Component-specific overrides

```css
justifi-checkout::part(button-primary) {
  background-color: #0d3b66;
  color: #fff;
}

justifi-business-form::part(section-header) {
  text-transform: none;
  font-weight: 600;
}
```

For the full matrix of exposed parts review `packages/webcomponents/src/styles/parts.ts` in this package.

## Accessibility + localization

- Labels and inputs are wired with `aria-describedby` out of the box.
- Error summaries focus the first invalid field and announce through `role="alert"`.
- Locale-aware helpers format currency, addresses, and phone numbers when you pass `locale`.
