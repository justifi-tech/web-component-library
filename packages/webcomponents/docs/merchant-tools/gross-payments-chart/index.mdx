## Overview

Component to render chart displaying last 30 days of gross payment data.

## Usage

### Chart monthly volume

<pre><code className="language-html">{`<justifi-gross-payment-chart
  account-id="acc_123"
  auth-token="wct_reporting"
  interval="month"
  range="12m"
/>`}</code></pre>
