## Overview

Component to render chart displaying last 30 days of gross payment data.

## Usage

<UsageSnippet
  title="Chart monthly volume"

  account-id="acc_123"
  auth-token="wct_reporting"
  interval="month"
  range="12m"
/>`}
  language="html"
/>
