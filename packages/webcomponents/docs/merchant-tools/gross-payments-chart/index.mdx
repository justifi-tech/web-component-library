## Overview

`<justifi-gross-payment-chart>` provides a lightweight chart that plots payment volume over time. Use it on dashboards to highlight trends without standing up bespoke charting infrastructure.

## Usage

<UsageSnippet
  title="Chart monthly volume"
  code={`<justifi-gross-payment-chart
  account-id="acc_123"
  auth-token="wct_reporting"
  interval="month"
  range="12m"
/>`}
  language="html"
/>

## Theming & Layout

- Customize colors via CSS properties such as `--justifi-chart-line-color`, `--justifi-chart-fill-color`, and `--justifi-chart-axis-color`.
- Provide a minimum width of 320px; the chart is responsive and scales up to fill its container.

## Accessibility

- Data points become focusable elements so keyboard users can inspect values with arrow keys.
- Tooltips use `aria-live="polite"` to announce the current amount and date.
- Provide descriptive `aria-label` text summarizing the selected interval/range.
