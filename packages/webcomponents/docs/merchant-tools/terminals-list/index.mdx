## Overview

Component to render a formated list of terminals for the requested account.

## Usage

<UsageSnippet
  title="Terminal inventory"
  code={`<justifi-terminals-list
  account-id="acc_123"
  auth-token="wct_terminals"
  filters='{"status":"online"}'
/>`}
  language="html"
/>
