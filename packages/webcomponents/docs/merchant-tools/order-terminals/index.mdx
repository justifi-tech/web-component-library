## Overview

`<justifi-order-terminals>` streamlines the hardware ordering process by surfacing compatible devices, subscription pricing, shipping info, and confirmation states. Use it in consoles where merchants manage their in-person experiences.

## Usage

<UsageSnippet
  title="Order a reader"
  code={`<justifi-order-terminals
  account-id="acc_123"
  auth-token="wct_terminals"
  shipping-contact="contact_456"
/>`}
  language="html"
/>

### Events

- `submitted`: Fires when an order is placed; includes order ID and device list.
- `error-event`: Emits network or validation errors.

## Theming & Layout

- Customize card surfaces with `--justifi-card-surface` and `--justifi-card-border`.
- Each device card is 320px wide; on desktop the component displays a grid, while on mobile it stacks vertically.

## Accessibility

- Product cards behave like checkboxes with `aria-checked` so assistive tech understands which devices are selected.
- Confirmation view announces with `role="status"` to confirm success.
