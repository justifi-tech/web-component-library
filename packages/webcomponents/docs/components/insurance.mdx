## Overview

`<justifi-insurance>` promotes insurance products (e.g., season interruption coverage) anywhere in your experience. It exposes CTA hooks so you can launch deeper flows or simply capture consent.

## Usage

<UsageSnippet
  title="Inline upsell"
  code={`<justifi-insurance
  policy-id="season_interrupt"
  premium="9.99"
  coverage-summary="Protect revenue when weather forces closures."
  default-selected
/>`}
  language="html"
/>

## Theming & Layout

- Customize using CSS props like `--justifi-insurance-surface`, `--justifi-insurance-border`, and `--justifi-insurance-spacing`.
- The card works well between 280px and 420px wide; on mobile it should span the full width with comfortable padding.
