## Overview

`<justifi-dispute-management>` gives support teams everything they need to triage disputes: evidence upload, status tracking, deadlines, and messaging. Embed it wherever your merchants expect to manage chargebacks.

## Usage

<UsageSnippet
  title="Open a dispute detail view"

  dispute-id="dp_456"
/>`}

### Events

- `evidence-uploaded`: Fires when users add files or notes.
- `status-changed`: Indicates dispute submission or withdrawal.
- `error-event`: Captures upload or API issues.

## Theming & Layout

- Adjust the card surfaces and callouts using CSS props such as `--justifi-card-surface`, `--justifi-card-border-color`, and `--justifi-alert-info-background`.
- Keep layouts within 640px to maintain readable evidence lists; on mobile the layout collapses into accordions.

## Accessibility

- Evidence lists are ordered lists with descriptive `aria-label` text.
- File inputs include `aria-describedby` text that restates acceptable formats and size limits.
- Status update banners use `role="status"` so assistive tech announces them immediately.
