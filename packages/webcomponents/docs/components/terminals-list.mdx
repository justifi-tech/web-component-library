## Overview

`<justifi-terminals-list>` presents terminals deployed in the field along with health, assigned location, and software version. Operations teams use it to troubleshoot and filter devices.

## Usage

<UsageSnippet
  title="Terminal inventory"
  code={`<justifi-terminals-list
  account-id="acc_123"
  auth-token="wct_terminals"
  filters='{"status":"online"}'
/>`}
  language="html"
/>

### Events

- `click-event`: Row click event that surfaces `terminal_id`.
- `filter-changed`: Emits when the built-in filters change.

- Table styling aligns with other reporting components; adjust CSS props like `--justifi-table-header-bg`.
