## Overview

`<justifi-bank-account-form>` collects ACH details inside Modular Checkout. It inherits auth/account context from the parent and emits tokens back through the checkout submission pipeline. Use it when you need to offer bank transfers alongside cards or alternative rails.

  title="Bank rail inside Modular Checkout"

### Events

- Emits the same `submit` / `error` events as other modular rails via the parent checkout; no direct listeners required.

### Public methods

1. `focus()` – Focuses the first hosted field (routing number).

## Theming & Layout

- Share the same CSS props as other forms: `--justifi-form-surface`, `--justifi-form-border-color`, and `--justifi-form-spacing`.
- Keep width between 320px and 520px for readability; on mobile it stacks inputs vertically with built-in spacing.
